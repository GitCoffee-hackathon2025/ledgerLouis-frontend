## 🚧 Desenvolvimento

Como a API ainda está em desenvolvimento, acesse a branch `dev`:

```bash
# vá para dev
git checkout dev
# Para usar uma versão mais estável:
git checkout 04eff5dcef294a6943d543e3961576d3b6e7ebe7
npm i
```

Se o backend estiver rodando em http://localhost:3000/ apenas execute o frontend


```bash
npm run dev
```
